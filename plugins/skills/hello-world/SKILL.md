# Skill: hello-world

## Description

A simple hello world skill for demonstration. This is the **staging/beta** version with new features.

## When to Use

Use this skill when you want to demonstrate the basic functionality of the marketplace system, or when you need the new beta features.

## New Features in v2.0.0-beta

- **Enhanced Greeting**: More colorful and informative messages
- **Environment Detection**: Automatically detects which marketplace branch is being used
- **Debug Mode**: Shows additional debug information

## Instructions

1. Greet the user with an enhanced friendly message
2. Show the current environment information including branch name
3. Display new feature announcements
4. Confirm the skill is working properly in staging mode
